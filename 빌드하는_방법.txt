APK용 빌드 프로필을 추가해뒀어요. 이제 Expo 쪽에서는 preview 프로필로 바로 APK를 만들 수 있고, 정식 배포용은 production으로 AAB 중심으로 가져가면 됩니다.

이 프로젝트는 이미 Expo 앱이라 EAS Build로 APK 만들면 됩니다. 제가 preview용 APK 프로필을 eas.json 에 추가해뒀어요.

바로 하는 순서는 이거예요.

npm install
npx eas-cli login
npx eas-cli build:configure
npx eas-cli build -p android --profile preview

그러면 Expo 서버에서 Android apk를 빌드해주고, 끝나면 다운로드 링크가 나옵니다.
