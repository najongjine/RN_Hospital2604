pollination api 에서 쓸 모델 이름은
openai-fast.

사용할 api는
post/v1/chat/completions

curl https://gen.pollinations.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
